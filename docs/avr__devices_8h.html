Detailed Description

AVR Device Database Types and Function Prototypes.

This header defines the data structures and lookup functions for the AVR device database. The database maps device signatures to device-specific parameters needed for correct programming.

Device Signature Format:

  • Byte 0: Manufacturer (0x1E = Atmel/Microchip)
  • Byte 1: Flash size indicator
  • Byte 2: Specific device variant
